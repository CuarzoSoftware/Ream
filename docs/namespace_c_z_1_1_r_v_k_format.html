Detailed Description

DRM FourCC <-> VkFormat mapping.

Complements SK/RSKFormat (DRM <-> SkColorType) and GL/RGLFormat (DRM <-> GL). Only formats meaningful to the Vulkan backend (and understood by Skia's Vulkan backend for render targets) are mapped. DRM formats are little-endian byte-order codes, so the mapping matches on in-memory byte layout.
