# 💬 Contact {#contact_page}

### Bugs

If you encounter any bugs, kindly open an [🐞 issue](https://github.com/CuarzoSoftware/Ream/issues).
Please provide details about the circumstances under which the bug occurs, your hardware
specifications, and any available debugging logs (see @ref environment_page).

### Support

If you have questions about Ream's API or internal behavior, please post them in the
[🤝🏻 Support](https://github.com/CuarzoSoftware/Ream/discussions/categories/support) section so
that everyone can benefit from the discussion.

### Suggestions and New Ideas

If you have suggestions for functionality to include in Ream or ideas to enhance its features,
please share them in the [💡 Ideas](https://github.com/CuarzoSoftware/Ream/discussions/categories/ideas)
section.
